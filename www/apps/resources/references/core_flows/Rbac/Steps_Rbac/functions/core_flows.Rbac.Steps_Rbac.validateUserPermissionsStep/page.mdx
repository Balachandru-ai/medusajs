# validateUserPermissionsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `validateUserPermissionsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

Validates that a user has access to all the policies they are trying to assign.
A user can only create roles and add policies that they themselves have access to.

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@medusajs/framework/workflows-sdk"
import { validateUserPermissionsStep } from "@medusajs/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = validateUserPermissionsStep({
      "actor_id": "{value}"
    })
  }
)
```
