# createTranslationsStep - Medusa Core Workflows Reference

This documentation provides a reference to the `createTranslationsStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step creates one or more translations.

## Example

```ts
const data = createTranslationsStep([
  {
    reference_id: "prod_123",
    reference: "product",
    locale_code: "fr-FR",
    translations: { title: "Produit", description: "Description du produit" }
  }
])
```
