# removeUserRolesWorkflow - Medusa Core Workflows Reference

This documentation provides a reference to the `removeUserRolesWorkflow`. It belongs to the `@medusajs/medusa/core-flows` package.

This workflow removes roles from users.
Supports two modes:
- Remove multiple roles from a single user: &#123; user\_id, role\_ids &#125;
- Remove multiple users from a single role: &#123; user\_ids, role\_id &#125;
It validates that the actor has all the policies from the roles being removed.
